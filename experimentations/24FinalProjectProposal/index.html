Bullet Heaven


Description


The aim of this project is to visulize a song, but this time add a layer of interactivity. I aim to visualize a song as if it was a boss battle from a typical bulet hell game. For a long time now, I've been trying to visualize music in this manner. I've been trying to see the many layers of instrumentation as bullet hell attacks. With my knowledge of cooding so far, I think I can attempt to experiment with this sort of visualization. I'm excited to see how it turns out.
